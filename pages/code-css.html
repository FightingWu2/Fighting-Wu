CSS 动画与性能优化

🎨 动画让网页更灵动，性能优化让体验更丝滑。

✨ 动画类型

  • 过渡（transition）
  • 关键帧动画（@keyframes）
  • transform/opacity 动画

🚀 性能建议

  • 尽量使用 transform/opacity
  • 避免频繁 layout/repaint
  • 合理使用 will-change

🧑‍💻 动画代码示例

.animated {
  transition: transform 0.5s cubic-bezier(.4,2,.6,1);
  will-change: transform;
}
善用硬件加速，减少重排重绘。

📦 动画库推荐

  • Framer Motion
  • GSAP
  • TailwindCSS 动效

🛠️ 实战技巧

  • 动画时长建议 0.3~0.7s
  • 交互动画要有节奏感
  • 复杂动画建议用 JS 控制
